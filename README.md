## 👋 Welcome to netbird 🚀

Netbird - WireGuard-based VPN mesh

## 📋 Description

WireGuard-based VPN mesh

## 🚀 Services

- **app**: Netbird (`netbirdio/netbird:latest`)

## 📦 Installation

```shell
composemgr install netbird
```

## 🔧 Configuration

```shell
TZ=America/New_York
BASE_HOST_NAME=netbird.example.com
```

## 🌐 Access

- **Netbird**: http://localhost:33073

## 📂 Volumes

- `./rootfs/data/netbird` - Application data

## 🔐 Security

- Change default passwords
- Configure HTTPS with reverse proxy
- Regular backups
